Detailed Description

Parsing the comandstring.

Supported Syntax Comand( Arg1, Arg2, ..., ArgN ) Whitespaces get removed, including the ones in the arguments also the Ending parenthesis (depreciated: CommandLine works also without ending parenthesis)

Parsing of the command.
